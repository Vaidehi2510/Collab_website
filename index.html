06 Known issues

What to know
before you start.

Release transparency

This Collab release is a major step forward, and there are three known limitations we want you to have up front.

Explore release highlights ↓
  1. 01 Issue #291

    Windows local models require an NVIDIA GPU.

    Collab cannot run local models on Windows machines without an NVIDIA GPU. Hub-deployed models remain available.

    Windows · Local models
  2. 02 Issue #539

    Enabling several connectors takes an extra step.

    The Connections menu closes after each connector toggle. Reopen it to enable the next one—your selections are still saved.

    Connectors · All platforms
  3. 03 Issue #249

    Long Hub replies may need a refresh on Windows.

    A longer response might not render in the active chat until Collab is reopened. The response is saved and remains in History.

    Windows · Hub chat
